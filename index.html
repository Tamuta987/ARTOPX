Cycle tir
Flux terrain : garder/changer position, créer un objectif par rayonnement direct, puis calculer la correction sur point moyen.
Pour iPhone, les étapes sont séparées au lieu de popups bloquantes en boucle. Utilise l’onglet Tir pour saisir les valeurs, puis les boutons du cycle pour enchaîner.
